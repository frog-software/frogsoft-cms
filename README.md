## 启动项目

1. 安装 `Docker` 和 `docker-compose`，确保良好的互联网访问
2. 复制 `.env.example` 为 `.env`，复制 `.env.prod.example` 为 `.env.prod`
3. 运行 `./runner.sh start deploy` 即可，在 `user.localhost` 和 `admin.localhost` 测试用户端和管理端
